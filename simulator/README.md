# IoT Sensor Simulator

Симулятор датчиков окружающей среды для тестирования системы мониторинга.

## Описание

Симулятор создает 5 виртуальных датчиков в разных точках Павлодара:
- Центральный парк (низкое загрязнение)
- Промзона НПЗ (высокое загрязнение)
- ЖК Достык (умеренное загрязнение)
- Автовокзал (высокое загрязнение от транспорта)
- Мобильная станция (переменное загрязнение)

## Запуск

```bash
# Установите зависимости
pip install -r requirements.txt

# Запустите симулятор
python sensor_simulator.py
```

## Особенности

- Реалистичные данные с учетом времени суток
- Rush hour эффекты (8-9 утра, 17-19 вечера)
- Случайные всплески загрязнения (5% вероятность)
- Автоматическая регистрация датчиков
- Отправка данных каждые 30 секунд

## Параметры

Изменить частоту отправки данных можно в переменной `interval` (секунды).
